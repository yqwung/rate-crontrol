3.3 ABR

3.3 ABR

平均比特率（Average Bitrate）

平均比特率也称目标比特率，ABR 不是恒定比特率，虽然它在技术上是一种 CBR 模式，但它并不比恒定比特率模式好多少。应该避免使用此模式。
